## Projects

### Awam Connect KP
A civic Android app connecting citizens of KP with their local 
government. Features member registration, complaint submission, 
democratic city head voting system, and full admin control panel. 
Built in 2.5 hours at a competition and won 3rd place at the 
3rd International Student Expo Islamabad.
- Tech: Android, Firebase, Java

### Amaan Safety App
A women safety application for Pakistan built with Flutter. 
Features voice-activated emergency alerts using a panic keyword, 
shake detection via accelerometer, automated SMS and Email dispatch 
with real-time location data, and quick-dial access to Police, 
Ambulance, and Women Protection services.
- Tech: Flutter, Dart, Java, Google Maps API
- GitHub: https://github.com/waqaramd55/Amaan-Safety-App

### Healix — AI Medical Assistant Backend
A REST API backend with JWT authentication, medication tracking, 
BMI and BMR health metrics calculation, AI-driven chat, and 
SQLite/PostgreSQL database integration. Built as a personal 
project to explore AI integration in healthcare applications.
- Tech: Python, FastAPI, PostgreSQL, SQLite, AI/ML
- Demo: https://drive.google.com/file/d/1TFzMgd2kDyA9Ni4c4p5sZeLXlH3kqZNm/view

### Teerop FoodOS
A Pakistan-focused restaurant logistics platform concept. 
Flat PKR 10,000/month subscription model replacing per-order 
marketplace commissions. Features AI-powered dispatch engine, 
chatbot ordering, live GPS tracking, and demand forecasting.
- Tech: System Design, FastAPI, Flutter, AI
- Status: Concept Project

## Getting Started

Clone the repository and run locally:

```bash
git clone https://github.com/abdulrehmanzahid160/portfolio.git
cd portfolio
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
